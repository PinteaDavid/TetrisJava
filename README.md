# Tetris Java

A desktop Tetris game built in Java using Swing.

## Features

- Tetromino movement and rotation
- Collision detection
- Line clearing
- Score and level system
- Next piece preview
- Pause and game over states
- Sound support

## Technologies

- Java
- Java Swing
- Object-Oriented Programming

## Controls

- Left Arrow: move left
- Right Arrow: move right
- Down Arrow: move down faster
- Up Arrow: rotate piece
- Space: pause / resume

## Project Structure

- `src/main` - main game logic, window, input handling, sound
- `src/mino` - tetromino and block classes
- `res` - audio resources

## How to Run

Recommended way:

1. Open the project in IntelliJ IDEA
2. Run `main.Main`

## About

This project was made for practice with Java, Swing, object-oriented programming, and game logic.
